# Monster Type

15問で、あなたの中のモンスターを召喚する性格診断Webゲームです。

## Public URL

https://monster-type.dimension0122.workers.dev/

## Features

- 15問の性格診断
- モンスタータイプ診断
- 属性タイプ
- レア度
- モンスター相性
- 詳細レポート
- 進化クエスト
- 結果シェア

## Tech Stack

- React
- TypeScript
- Vite
- Cloudflare Pages / Workers

## Development

npm install

npm run dev

## Build

npm run build
